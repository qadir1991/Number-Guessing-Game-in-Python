🎯 Number Guessing Game in Python
📌 Description

This is a simple Python-based number guessing game where the computer randomly selects a number, and the player tries to guess it.

The program gives helpful hints like:

📉 Too Low

📈 Too High

until the correct number is guessed.

This project is perfect for beginners to practice:

Loops

Conditional statements

User input handling

Basic Python logic

🚀 Features

Random number generation

User-friendly hints

Unlimited attempts

Simple and clean code

Beginner-friendly

🛠️ Requirements

Python 3.x

▶️ How to Run

Clone this repository:

git clone https://github.com/your-username/number-guessing-game.git

Navigate to the project folder:

cd number-guessing-game

Run the program:

python game.py
